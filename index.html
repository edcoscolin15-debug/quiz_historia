📖 Guia Prático de uso

Este roteiro foi desenhado para que você domine todas as funcionalidades, garantindo que suas aulas sejam dinâmicas e seus dados estejam sempre seguros.

1. Painel Inicial (Menu Principal)

Ao abrir o arquivo no seu navegador (Chrome ou Edge), você encontrará quatro botões principais:

  • 🎮 Iniciar Quiz Show: Acesso à lista de temas disponíveis para começar a partida.
  • 🛠️ Editor de Conteúdos e Perguntas: Acesso ao "Motor de Criação" (Editor), onde você gerencia o banco de dados.
  • 📥 Exportar Banco de Questões: Painel para gerar seus arquivos de backup (escolha "Tudo" ou apenas um tema específico).
  • 📤 Importar Arquivo do Quiz: Carrega arquivos .json que você criou ou salvou anteriormente.

2. O Ambiente do Jogo (Em Ação)

Ao iniciar uma partida no seu tablet/celular Android, você verá o ambiente de jogo:

  • Placar Dinâmico: Localizado à esquerda, use os botões "+ Ponto" para registrar o progresso dos times 1, 2 ou 3 em tempo real.
  • Cronômetro (60s): O tempo começa a contar automaticamente. Se chegar a zero, o tempo esgota e o sistema bloqueia as seleções.
  • Alternativas: Escolha entre 4 opções. O sistema destaca em verde se acertar ou em vermelho se errar.
  • Recursos Auxiliares:
    • Mostrar Dica: Revela uma dica exclusiva para ajudar os alunos.
    • Explicação Histórica: Aparece automaticamente após a seleção da alternativa, funcionando como um gabarito comentado.
  • Navegação: Use os botões "Anterior" e "Próxima" para transitar entre as questões do tema selecionado.

3. Editor de Conteúdos e Perguntas

Aqui é onde você mantém o seu banco de questões sempre atualizado:

  1. Criar Novo Tema: Digite o nome no campo "Criar Novo Tema Histórico" e clique em "Criar". O sistema salvará automaticamente.
  2. Filtragem: Use o filtro no topo para isolar um tema específico. Isso facilita a edição sem precisar navegar por todas as perguntas.
  3. 🗑️ Excluir Tema: Se um tema não for mais necessário, selecione-o no filtro e clique em "🗑️ Excluir Tema". O sistema solicitará confirmação antes de apagar o conteúdo e todas as suas perguntas associadas.
  4. Editar/Adicionar Perguntas: Selecione uma questão para editar ou clique em "Criar Nova Pergunta do Zero". Preencha o enunciado, as 4 alternativas, indique a correta e insira a dica e explicação.
  5. 💾 Salvar: Sempre clique em "Salvar Pergunta" para garantir que as alterações sejam gravadas na memória do seu dispositivo.

4. Exportação e Importação (.json)

Esta é a sua ferramenta de segurança:

  • Exportar: Escolha se quer baixar o Banco Completo ou apenas um Tema específico. O sistema gerará um arquivo .json e o tablet fará o download automático para a pasta de arquivos do sistema.
  • Importar: Se você for para outra escola ou trocar de tablet, basta clicar em "Importar", selecionar o ficheiro .json e o sistema mesclará as perguntas automaticamente.

📱 Usabilidade no Android (Tablet ou Celular)

O Quiz de História Pro foi otimizado para a experiência móvel:

  • Offline: Uma vez que o ficheiro .html esteja salvo no dispositivo, não é necessária internet para jogar, editar ou importar questões.
  • Salvamento Automático (LocalStorage): O Android armazena todas as suas alterações na memória interna do navegador. Se você fechar o navegador, ao retornar, seu progresso de edição estará intacto.
  • Toque Preciso: A interface responsiva adapta-se ao tamanho da tela. Em tablets, o layout mantém o placar fixo à esquerda; em celulares, ele ajusta-se para garantir que os botões de resposta sejam grandes e fáceis de tocar.

Dica de mestre: Mantenha uma cópia dos seus arquivos .json no Google Drive. Assim, se precisar trocar de tablet, terá o seu banco de questões sempre disponível em poucos segundos.

⚠️ ATENÇÃO

O arquivo .html é só o motor do jogo (a estrutura, o visual, as regras). As perguntas que você cria ficam na memória do navegador. E os arquivos .json que você exporta são o seu verdadeiro banco de dados (os cartuchos de jogo) que você guarda para não perder nada, então após criar um tema e os desafios, exporte para sua segurança e importe quando conveniente.

Se você quiser que uma pergunta nova faça parte do arquivo .html para sempre (sem precisar importar), aí sim você precisaria abrir o .html no Bloco de Notas e digitar o código lá dentro. Mas, com o sistema de exportar/importar .json que criamos, você não vai precisar mexer em código.

Se preferir agilizar a criação de conteúdos (temas) e os desafios, pode solicitar para a IA montar em arquivo JSON (DICA: envie um modelo exportado para a IA analisar a estrutura do código JSON). Após ela montar, basta fazer o download (ou copiar o código e colar em um Bloco de Notas ou NotePad++ e salvar com o nome do "conteúdo.json" seguindo o formato, exemplo (IDADE_MEDIA.json).

Lembre de solicitar os critérios:

  • Dificuldades misturadas (fácil, média e difícil).
  • Todas valem 1 ponto. (ou pontuar de acordo com o nível de dificuldade, porém toda dificuldade valer 1 torna mais justo)
  • Ordem totalmente aleatória.

🤖 Prompt para IA

"Atue como um especialista em pedagogia e desenvolvimento de jogos educativos. Com base na estrutura JSON que estamos utilizando em nosso projeto, crie um novo arquivo JSON para um quiz educacional sobre o tema: [INSERIR O TEMA AQUI]. Regras rigorosas para a criação: 1. Mantenha exatamente a mesma estrutura de chaves do nosso projeto. Cada objeto dentro da lista perguntas deve obrigatoriamente conter: - conteudo: (String) O tema da pergunta. - pergunta: (String) O enunciado. - nivel: (String) 'Fácil', 'Médio' ou 'Difícil'. - pontos: (Number) Valor da pontuação (ex: 1). - correta: (Number) O índice da alternativa correta (0 a 3). - dica: (String) Uma breve dica. - explicacao: (String) Breve explicação pedagógica. - alternativas: (Array com 4 Strings). 2. Crie [INSERIR NÚMERO DE PERGUNTAS] perguntas equilibradas entre os níveis. 3. Certifique-se de que o JSON esteja validado e sem erros de sintaxe, pronto para ser lido pelo localStorage do nosso jogo. Apresente o código final dentro de um bloco de código."

Dicas para obter os melhores resultados:

  • Seja Específico: Quanto mais detalhes você der no campo [INSERIR O TEMA AQUI] (ex: "A Revolução Industrial" ou "O Período Regencial Brasileiro"), melhor será o conteúdo das perguntas.
  • Adaptação do Público: Se você quiser que o quiz seja para uma faixa etária específica (ex: Ensino Fundamental II ou Médio), adicione essa instrução ao prompt, e eu ajustarei a linguagem e a complexidade das perguntas para você.
